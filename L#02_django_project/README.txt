Project:
	A group of applications and files is called a Django Project 

Steps to follow, python must be installed first.
. Install django by writing this command on command prompt --> pip install django
. Create a project by opeing the terminal in the working directory in vs code 
. Syntax to create a project is --> django-admin startproject projectname(course here)
. to run the project, execute this line of code --> python manage.py runserver
. now copy the path like --> http://127.0.0.1:8000/ and paste it in the web browser
